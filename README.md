## 📖 Overview

Smart Tab Manager is a Chrome Extension designed to make tab management simple and efficient.

Whether you're a student, developer, researcher, or multitasker, opening dozens of tabs can quickly become overwhelming. Smart Tab Manager helps you stay organized by providing an intuitive interface to view, search, organize, and manage your browser tabs—all from one place.

---

## ✨ Features

- 📑 View all open tabs
- 🔍 Instantly search tabs
- 📌 Pin and unpin tabs
- ❌ Close individual tabs
- 🧹 Close duplicate tabs
- 💤 Suspend inactive tabs *(Upcoming)*
- 📂 Group tabs by domain *(Upcoming)*
- ⭐ Save tab sessions *(Upcoming)*
- 🌙 Clean and modern UI
- ⚡ Fast and lightweight

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/Giribalajichandrashekar/Smart-tab-manager-extension.git
```

### Open Chrome Extensions

```
chrome://extensions/
```

### Enable

- Developer Mode

### Click

```
Load unpacked
```

### Select

```
Smart-tab-manager-extention
```

The extension is now ready to use.
